# milwaukee-mental-stress-heatmap
A Heatmap of Mental Stress in Milwaukee According to CDC
Overview
This repository contains a geospatial visualization tool designed to map mental stress indicators across the city of Milwaukee. By utilizing a heatmap layer, this project provides a data-driven perspective on geographic trends in community well-being, allowing researchers, city officials, and healthcare providers to identify areas that may benefit from increased mental health resources and support.

Features
Dynamic Heatmap Layer: Visualizes the intensity and distribution of stress-related data points across Milwaukee neighborhoods.

Interactive Interface: Users can zoom, pan, and hover over specific areas to gain a better understanding of the local geographic context.

Web-Based Delivery: The visualization is contained within a standalone HTML file, making it easily accessible via any modern web browser without the need for complex backend infrastructure.

Technical Stack
HTML5/CSS3: Structure and styling of the visualization interface.

JavaScript: Core logic for data rendering and map interactivity.

Mapping Library: Utilizes industry-standard libraries (such as Leaflet.js or OpenLayers) to render the underlying map tiles and heatmap gradient.
